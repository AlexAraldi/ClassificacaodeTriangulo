# Classificação de Triângulos

Este é um programa simples em C# que permite ao usuário inserir os comprimentos dos lados de um triângulo e determina sua classificação como equilátero, isósceles ou escaleno.


![](https://i.imgur.com/TdDAG2s.gif)



## Como Executar o Programa
1. Certifique-se de ter o .NET SDK instalado em sua máquina.
2. Compile e execute o programa com os seguintes comandos no terminal:
   ```sh
   dotnet run
   ```
3. Informe os valores dos lados do triângulo conforme solicitado.
4. O programa validará os lados e exibirá a classificação do triângulo.
5. O usuário pode escolher continuar inserindo novos valores ou sair.

## Classificação dos Triângulos
O programa determina se o triângulo é:
- **Equilátero**: Todos os lados são iguais.
- **Isósceles**: Dois lados são iguais.
- **Escaleno**: Todos os lados são diferentes.

Se os valores informados não formarem um triângulo válido, o programa exibirá uma mensagem de erro.
